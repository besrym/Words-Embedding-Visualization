# Words Embedding & Visualization

# Installation
```
conda create -n we-venv -y python=3.7 && conda activate we-venv
pip install -r requirements.txt
```

# Usage

Insert project path e.g. : 
```
/Users/your_name/Desktop/WordsEmbedding
```
then run the script:
```
python main.py
```
